## Commands

### Beginning Commands
```sh
yarn install
```

```sh
yarn add firebase
```

### Development Mode
```sh
gatsby develop --host localhost --port 8000
```

### Build Application
```sh
gatsby clean
```
```sh
gatsby build
```

### Deploy Application
```sh
yarn add -g firebase-tools
```
```sh
firebase init
```
```sh
firebase deploy
```
